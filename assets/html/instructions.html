User Instructions

The RSNA DICOM Anonymizer is a general-purpose stand-alone de-identification utility.
It imports DICOM images either via network transfers or from files from external storage. After import it de-identifies and stores them in a local file directory. Anonymized images can be exported to a DICOM server or an AWS S3 Object store such as the RSNA Imaging Archive.
Full documentation to be completed after BETA testing by users familiar with JAVA Anonymizer (who are familiar with the working principles).
